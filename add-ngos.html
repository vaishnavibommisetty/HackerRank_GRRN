🔧 GRRN Database Setup

Add NGOs to your Firebase database

🔧 Manual Setup (If button doesn't work)

Follow these steps in Firebase Console:

  1. Go to your Firebase Console
  2. Navigate to Realtime Database → Data
  3. Click the "+" icon next to your project name
  4. Name the new node: ngos
  5. Click "+" again to add values
  6. Add each NGO as a key with value true:
Red Cross: true
Food Bank: true
Health Aid: true
Community Kitchen: true
Save the Children: true
